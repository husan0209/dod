# docs/security.md

## Политика безопасности DOD Platform

### 1. Политика паролей
- Минимум 8 символов
- Сочетание букв, цифр, символов
- Запрет на распространенные пароли

### 2. 2FA (обязательная для админов)
- Google Authenticator для админов
- Email код для пользователей

### 3. IP whitelist
- Админ-панель ограничена по IP
- Webhook Telegram ограничен по IP

### 4. HTTPS / HSTS
- Все соединения через HTTPS
- HSTS preload включен

### 5. Rate limiting
- 5 попыток логина в 15 минут
- 30 req/s для API
- 3 SMS в 10 минут

### 6. Обработка инцидентов
- Мониторинг через Prometheus
- Алерты через Telegram
- Post-mortem анализ

### 7. Ротация секретов
- Ежемесячно менять ключи
- Автоматическая ротация токенов

### 8. Аудит доступа
- Логи всех действий админов
- Аудит входов пользователей
